# tinybiome
Cell eat cell world out there!


Build the binaries:
If you don't have gb, [install it](http://getgb.io/docs/install/)
````
gb build
````

To run the server:
````
./bin/master
````

To run the nodes:
````
./bin/node
````
Optionally, provide --port N

Then connect via http to the master to start playing.
